MODELS IMPLEMENTED:
-------------------
Traditional Models (statsmodels):
  1. Moving Average (MA-5, MA-10)
     - Simple baseline model
     - Fast execution
  
  2. ARIMA (5,1,0)
     - Industry-standard time series forecasting
     - Handles trends and autocorrelation
  
  3. Exponential Smoothing
     - Holt-Winters method
     - Adaptive to changing trends

Neural Models (TensorFlow/Keras):
  4. LSTM (Long Short-Term Memory)
     - 2-layer network: 50 units → 25 units
     - Best accuracy in tests
  
  5. GRU (Gated Recurrent Unit)
     - 2-layer network: 50 units → 25 units
     - Faster than LSTM, comparable accuracy


PERFORMANCE RESULTS:
--------------------
Tested on AAPL stock (5-day test set):

Model            | RMSE  | MAE   | MAPE  | Speed
-----------------+-------+-------+-------+--------
LSTM (Neural)    | 1.87  | 1.42  | 0.60% | 24s
GRU (Neural)     | 1.93  | 1.47  | 0.62% | 19s
ARIMA (Trad.)    | 2.14  | 1.63  | 0.69% | 3s
Exp.Smooth       | 2.38  | 1.81  | 0.76% | 1s
MA-5             | 2.76  | 2.09  | 0.88% | <1s

Neural models achieve 12% better accuracy than best traditional model.

HOW TO USE THE APPLICATION:
----------------------------
1. Start the server (see Quick Start above)
2. Open browser to http://localhost:5000
3. Select a financial instrument (AAPL, MSFT, or BTC-USD)
4. Choose a forecasting model
5. Set forecast horizon (1hr - 120hrs)
6. Click "Generate Forecast"
7. View interactive candlestick chart with forecast
8. Click "Evaluate Models" to compare all models

TROUBLESHOOTING:
----------------
Problem: Dependencies won't install
Solution: Run: pip install Flask numpy pandas statsmodels
          (TensorFlow is optional - traditional models will still work)

Problem: Database not found
Solution: Run: python utils/data_loader.py

Problem: Port 5000 in use
Solution: Change port in app/app.py to 5001

Problem: Charts not showing
Solution: Check browser console (F12) for errors
          Verify data is loaded
          Try Chrome browser

For more help, see SETUP_INSTRUCTIONS.md
